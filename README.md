# HR-Analytical-Dashboard
This project focuses on analyzing employee data to uncover insights related to attrition, workforce distribution, and employee demographics. Multiple dashboards were developed to provide a comprehensive view of HR performance and support data-driven decision-making.

Key Insights:
- Identified attrition trends across departments, job roles, and demographics
- Analyzed employee distribution by age, gender, education, and marital status
- Evaluated job satisfaction levels and workforce composition

Dashboard Features:
- KPI tracking for employee count, attrition, active employees, and average age
- Department-wise and role-based attrition analysis
- Employee segmentation by age group, gender, and education
- Job satisfaction analysis across different roles
- Interactive filters and slicers for dynamic exploration

Data Processing & SQL:
- Extracted and analyzed HR data using SQL queries
- Calculated key KPIs such as attrition count, attrition rate, and active employees
- Performed aggregations and segmentation by department, education, and gender

Tools & Technologies:
- Power BI
- Tableau
- Excel
- SQL
